| **:%:h/FILENAME**          | **:sp**               | **tab + h/l** |               | **:InstantMarkdownStop**    |
| -> new File in current dir | -> split horizont     | -> switch tab |               | -> Stop Preview             |
|                            | **:vsp**              |               |               ||
|                            | -> split vertical     |               |               |
|                            | **Ctrl-w + h/j/k/l**  |               |               |
|                            | -> move between vies  |               |               |
